CSS-3 RADIENTS...!

In radial gradient color emerge from a single point and smoothly spread outward in a circular or elliptical shape rather than fading from one color to another in asingle direction as with linear gradients. The basic syntax of creating a radient can be givenwith

SYNTAX : radial-gradient(shape size at position, color-stop1, color-stop2, ...);

The arguments of the radial-gradient() function has the following meaning:

  1. position - Specifies the starting point of the gradient, which can be specified in units (px, em, or percentages) or keyword (left, bottom, etc).
  2. shape - Specifies the shape of the gradient's ending shape. It can be circle or ellipse.
  3. size - Specifies the size of the gradient's ending shape. The default is farthest-side.
